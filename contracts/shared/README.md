# contracts/shared/

Reusable assets for contract development. Not a deployable program.

- **`fixtures/`** — test data (addresses, token IDs, sample auction configs) shared across test suites.
- **`scripts/`** — Leo scripts and shell helpers for local testing workflows (e.g., seeding test state, running a multi-step auction flow end-to-end in the Leo sandbox).
- **`docs/`** — contract-level reference docs (struct field descriptions, invariant tables).
