# Creating an auction

Launching a token auction on Fairdrop uses an 8-step wizard. You can save a draft at any
point and return to it — progress is preserved in your browser session.

---

## Wizard steps

| Step | Name | What you configure |
|---|---|---|
| 1 | **Type** | Which auction format to use |
| 2 | **Token** | The token you are selling and the supply |
| 3 | **Pricing** | Price curve or raise target — unique to each type |
| 4 | **Timing** | Start block, end block, per-bid limits |
| 5 | **Gate & Vest** | Access restriction and token vesting schedule |
| 6 | **Referral** | Review referral economics (no input required) |
| 7 | **Metadata** | Name, description, logo, social links |
| 8 | **Review** | Final check before submitting on-chain |

---

## Before you start

- Connect your Aleo wallet (Leo, Puzzle, Fox, or Soter).
- Ensure you hold enough ALEO credits to cover the protocol fee and transaction fees.
- You must own (or be the admin of) the token you want to sell.
- Note the current block height — timing is set in block numbers, not wall-clock time.
  Each block is approximately 10 seconds.
