# Earning on Fairdrop

Anyone can earn ALEO credits on Fairdrop through three mechanisms — no special role required.

---

## Three earning actions

| Action | Who can do it | Reward |
|---|---|---|
| **Close an auction** | Anyone | Fixed closer reward per auction (set by protocol) |
| **Slash an unrevealed bid** | Anyone | Percentage of the slashed payment amount |
| **Collect referral commissions** | Referrers | Commission from protocol fee pool |

All earnings are paid in ALEO credits.

---

## Where to earn

Go to the **Earnings** page (`/earnings`). Three tabs correspond to the three earning actions.
